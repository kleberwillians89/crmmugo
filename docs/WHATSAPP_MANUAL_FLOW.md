# WhatsApp manual

O fluxo usa `wa.me` e sempre tem o cliente como destinatário. Telefone é normalizado para o padrão brasileiro, a mensagem é revisada e editável em modal, e somente então o navegador abre o WhatsApp. Abrir o link não significa mensagem enviada. O número configurado da Mugô é remetente futuro da Meta Cloud API e nunca substitui o destinatário.

Não há envio automático, token Meta, disparo em massa ou registro falso de entrega.
